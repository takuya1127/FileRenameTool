# FileRenameTool

Pythonで作成するファイル名一括変更ツールです。

## 概要

選択したフォルダ内のファイル名を一括で変更します。

例

IMG_001.jpg
IMG_002.jpg
IMG_003.jpg

↓

旅行写真_001.jpg
旅行写真_002.jpg
旅行写真_003.jpg

## 機能

- フォルダ選択
- ファイル名一括変更
- 連番付与

## 使用技術

- Python
- tkinter
